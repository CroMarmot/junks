handwriting
==

##这是啥？

一个用 js 的canvas实现的网页和手机端网页的手写板.....

##使用

参照demo()

`new R_handwriting();` 创建新的对象

`init()`初始化 

`bind_canvas(csspath)`绑定要使用手写功能的canvas，传入css路径

`bind_colorselect(csspath)`绑定颜色切换按钮，传入css路径，它们各自value参数为对应的颜色值

`bind_clear(csspath)`绑定清空按钮，传入css路径

`picbase64()`返回图片的base64编码
